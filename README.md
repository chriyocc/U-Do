# U-Do

 A macOS menubar app that rotates tasks from a to-do list.

## Features

- **Menubar Display:** Tasks are shown directly on the macOS menubar.
- **Automatic Rotation:** Tasks rotate every 5 seconds (Default) for easy tracking.
- **Task Management:** Add, delete, and set time interval between task.

## Requirements

- macOS

- Python 3.10 or later

- pyobjc-framework-Cocoa
